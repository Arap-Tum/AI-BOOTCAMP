# 🚀 AI for Content Creation (LIVE PRACTICAL SESSION)

## 🎯 Goal of This Session
By the end of this class, you will be able to:
- Create a professional poster using AI  
- Write captions, hashtags, and post content  
- Create a marketing/explainer video using AI avatars  
- Download and use your content in real life  

---

# 🟢 PART 1: AI POSTER CREATION 

## 🎯 Project Choice:
👉 **Event Invitation Poster**

(You are creating a real event poster like: church event, business event, wedding, or seminar)

---

## 🧠 STEP 1: Generate Poster Content (AI)



## ✍️ Prompt 1 (Content Creation)

> Act as a professional event planner. Create a clean and attractive event invitation details for a poster. Include:
- Event title  
- Date  
- Time  
- Venue  
- Short description  
Make it short, clear, and elegant.

---

## 🧠 STEP 2: Generate Poster Design

Now take the content into AI image generator.

---

## 🎨 Prompt 2 (Poster Design)

> Create a professional and elegant event invitation poster. Use modern typography, soft gradient background, and minimal design. Include the event details clearly. Make it visually attractive, clean, and suitable for social media sharing (Instagram, WhatsApp, LinkedIn).

---

## 📌 STEP 3: Review Poster

Check:
- Is it readable?  
- Is it clean?  
- Is information clear?  

---

# 🟡 STEP 4: Write Caption + Hashtags

## ✍️ Prompt 3 (Caption)

> Write a short, engaging social media caption for this event poster. Start with a strong hook, make it exciting, and include a call to action. Keep it simple and human.

---

## 🔖 Prompt 4 (Hashtags)

> Generate 10 relevant hashtags for promoting an event like this on social media.

---

## 📤 STEP 5: How to Post

### Tell students:
Post on:
- WhatsApp Status  
- Instagram Feed  
- Facebook  
- LinkedIn (if professional event)

### Include:
- Poster image  
- Caption  
- Hashtags  

---

### 💡 Teaching Point:
Content without posting is useless. Execution is the skill.
